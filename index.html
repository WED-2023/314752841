about me
Hello, my name is Harold Hyde Paine!

I'm someone who believes that staying active and creative keeps the spirit young. I’ve always found peace and inspiration in nature, which is why hiking is one of my favorite pastimes. There's nothing quite like reaching a summit and taking in a breathtaking view. Those same landscapes often end up on my canvas, as I enjoy capturing nature through painting. Art allows me to slow down and express the beauty I see in the world around me. On livelier days, you’ll often catch me with a cricket bat in hand, enjoying a friendly match. It’s a sport that keeps me moving and connected with others. I also have a soft spot for bowling — something about the sound of pins falling is just so satisfying. Whether I'm on a trail, holding a paintbrush, or sharing laughs over a game, I make the most of every moment. Life is a gift, and I try to live mine with joy, color, and a touch of adventure.!

Q: Explain 3 key differences between web 1.0, web 2.0 and web 3.0
A: web 1.0: also known as 'Static Web' referes to the early days of the World Wide Web
- Basic HTML
- No CSS
- User interactivity very limited
web 2.0: emerged in the 2000s and improved on user interactivity and is still in development
- Early days of CSS until now
- User generated content emerges
- Social media, networking
web 3.0: also known as 'Semantic Web' or 'Decentralized Web', not as defined as the last versions
- The use of AI and ML algorithms
- The potential for a decentralized using p2p technologies
Main differences emerge: user interactivity, web 1.0 relied heavily on static content, web 2.0 improved on that
web 3.0 seeks to take it a step further and making interactivity more intuitive
user generated content is also a major difference, early days had very minimal user generated content, web 2.0 we started seeing social media
emerge, web 3.0 tries to enhance the quality of content generated by users
technological advancments, to no CSS in web 1.0 to dynamic and designed web with CSS in web 2.0 and now with web 3.0 explores AI and block-chain technologies
